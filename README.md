# LS2526-batalha-naval-react
Trabalho Prático de Linguagens Script (2025/2026). Jogo "Batalha Naval Avançada" desenvolvido em React 19 para o ISEC.
A aplicação inclui mecânicas de gestão de combustível, limite de tempo por turno (15 segundos) e um radar inteligente. Desenvolvido exclusivamente com Componentes Funcionais e Hooks.
